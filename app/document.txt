# âœ… **Why `php artisan storage:link` is used**

### âœ”ï¸ **1. To access uploaded files from the browser**

When you upload images, PDFs, or any files, Laravel stores them in:

```

```

But this folder is **NOT publicly accessible**.

So Laravel creates a **symbolic link** from:

```

```

This makes your uploaded files accessible via browser URLs like:

```

```

---

# âœ… **2. Needed for file uploads, profile pictures, documents**

Example:

* User uploads profile photo
* Laravel stores it in `storage/app/public/profile_images`
* Browser tries to load `/storage/profile_images/user1.jpg`
* If link is NOT created â†’ **404 Not Found**

After running this:

```

```

It works âœ”ï¸

---

# âœ… **3. Symbolic Link Explanation (Simple)**

**Symbolic link** = Shortcut

Laravel just creates a shortcut folder:

```
public/storage
```

which points to:

```
storage/app/public
```

So when you access `/public/storage`, you are actually accessing `/storage/app/public`.

---

Whenever you:

* Clear public folder
* Delete the previous link
* Use file upload system for the first time
